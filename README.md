# RecepApp

Aplicación web para gestión de pedidos desarrollada con React.

## Funcionalidades

- Gestión de clientes
- Gestión de items
- Creación de pedidos
- Edición y eliminación de registros
- Formularios con validación
- Listados con acciones
- Cambio en el estado del pedido según sus etapas

## Tecnologías utilizadas
- React
- JavaScript
- CSS
- Node.js

## Estructura del proyecto
Clientes
Items
Pedidos

Cada módulo permite:
- Crear
- Listar
- Editar
- Eliminar

## Objetivo del proyecto
Este proyecto fue desarrollado para resolver la problemática de mi pequeña empresa de pastelería, la toma de pedidos y gestión de clientes. 
También se desarrolló como práctica para mejorar habilidades en:

- desarrollo frontend con React
- manejo de estado
- diseño de CRUD
- organización de componentes
